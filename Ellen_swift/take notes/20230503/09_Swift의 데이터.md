### 데이터 타입 표현방법

`var a:Int = 5`

- 타입에 관한 키워드는 모두 대문자로 시작한다.

### 주로쓰이는 데이터 타입

| 타입 | 설명 | 예시 |
| --- | --- | --- |
| Int | 정수. 8바이트 | 5, 7, -123, 0 |
| Double | 부동소수(15자리). 8바이트(64비트) | 3.14, -14.187 |
| String | 문자열. 쌍따옴표 사용(””) | "A”, “a”, “한” |
| Bool | 참/거짓 | true, false |
| 기타 | UInt, UInt64, Uint32 … |  |
- Int
- Double
- String
